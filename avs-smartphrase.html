Epic Paste Workflow
Steps

1. Open .xlsx in Excel (not Google Sheets)
2. Select all (Ctrl+A) → Copy (Ctrl+C)
3. Paste into Epic SmartPhrase editor or Patient Instructions
4. Verify formatting → Save SmartPhrase

Epic-Safe Formatting (verified 2026-02-10)

Confirmed: Bold, Italic, Bold+Italic, Blue font (phones), Red font (warnings), Thin borders, Gray fill (section dividers), Merged cells, Center alignment, Wrap text, 2-column layout

Avoid: Background highlighting on text, >2 columns, Complex nested merges, Images

Naming Convention

.wwavs[topic] — e.g., .wwavsiehpeducation · .wwavsradnetxray

Build Standards

Font: Arial · Body: 10pt · Headers: 11–14pt bold
Phones: Bold + Blue (#0000FF) · Warnings: Bold + Red (#FF0000)
Columns: ~48 char width · Row height: 85px for location blocks
Section dividers: Full-row merge, gray fill (#D9D9D9), bold
