Epic Paste Workflow
Steps

1. Open .xlsx in Excel (not Google Sheets)
2. Select all (Ctrl+A) → Copy (Ctrl+C)
3. Paste into Epic SmartPhrase editor or Patient Instructions
4. Verify formatting → Save

v2.0 Changes (Epic Compatibility)

Fixed: Each data field now occupies its own row with cell-level formatting — no more richText arrays. Merged cells only on headers/sections. Predictable row heights. No spacer rows.

Epic-safe: Bold, Italic, Blue font (phones), Gray fill (sections), Thin borders, Merged header rows, Center alignment, Wrap text, 2-column layout.

Avoid: Background highlighting on text, >2 columns, Complex nested merges, Images.

Naming Convention

.wwavs[topic] — e.g., .wwavsiehpeducation

Build Standards

Font: Arial · Body: 10pt · Headers: 11–14pt bold
Phones: Bold + Blue (#0000FF) · Sections: Gray fill (#D9D9D9)
Columns: ~48 char width · Row height: 16px per field line
